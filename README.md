# eGreen landing page

A one-page, brand-native landing page prepared for eGreen ahead of eMerge Americas 2026. Built to sit in front of Andrés, Rodrigo, and Gabriel with a specific hypothesis about why corporate offtake conversations stall, followed by the Feeld-shaped engagement that addresses it.

The page uses eGreen's own palette (mint #06E89F on white, near-black navy #001529 for headings and CTA band), Montserrat throughout, and imagery-free composition that echoes the cleaner side of egreen.com. No Eduba branding at the top. Eduba is acknowledged only in the footer, as specified.

Intended URL path: /for/egreen

## Files

- `index.html` — single-file landing page
- `styles.css` — brand-pure CSS, no JS
- `logo.svg` — eGreen primary mint logo (used in header and favicon)
- `logo-white.svg` — white variant, kept for reference
